Dynasty is a promise-based DynamoDB client for Node. Focus on code cleanliness.

Dynasty is open source software and is released under the developer and business-friendly MIT license.

Or grab the latest version on Github.

Usage

node

Easy to install in Node.js

  npm install dynasty

then in your app

  // Load Dynasty
  var Dynasty = require('dynasty');

  // Instantiate Dynasty so it can be used
